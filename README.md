grpc-entgo-cqrs-example

[gRPC](https://github.com/grpc/grpc-go) Go implementation of gRPC<br/>

* Mutations
    * CreateContact
    * UpdateContact
    * DeleteContact
* Queries
    * GetContact
    * ListContacts
